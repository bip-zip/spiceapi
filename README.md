# SpiceAPI 

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/bip-zip/spiceapi.git
   cd spiceapi

2. Install dependencies
   ```bash
   pip install -r requirements.txt

3. Apply migrations
   ```bash
   python manage.py migrate
